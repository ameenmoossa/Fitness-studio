# Fitness Studio Frontend

A comprehensive fitness studio management platform frontend designed to streamline class bookings, membership management, attendance tracking, payment processing, and studio operations through a modern and responsive user interface.

The platform provides fitness enthusiasts with a seamless experience for discovering classes, booking sessions, managing memberships, tracking attendance, and monitoring fitness activities. Administrators can efficiently manage schedules, instructors, memberships, payments, and overall studio operations through a centralized dashboard.

**Fitness Management • Class Booking • Membership Management • Attendance Tracking • React.js • Studio Operations**

## 🚀 Installation

### 1. Clone Repository

```bash
git clone <repository-url>
cd Fitness-Studio-Frontend
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

```env
VITE_API_URL=http://localhost:5000/api
```

### 4. Run Development Server

```bash
npm run dev
```

Application runs on:

```text
http://localhost:5173
```
